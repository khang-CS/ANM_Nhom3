# Bài tập lớn An ninh mạng - Nhóm 03

## Đề tài

Nghiên cứu, xây dựng mô hình mạng an toàn cho Công ty CP kỹ thuật Hòa Phát.

## Thành viên nhóm

- Nguyễn Văn Hoàng
- Ngô Văn Khang (Nhóm trưởng)
- Bùi Văn Quang
- Nguyễn Văn Thủy
- Vũ Thế Vượng

## Nội dung thực hiện

- Nghiên cứu tổng quan về an ninh mạng.
- Tìm hiểu kiến thức cơ sở về mạng máy tính, an toàn thông tin và tấn công mạng.
- Nghiên cứu mô hình mạng an toàn trong doanh nghiệp.
- Xây dựng mô hình mạng an toàn cho Công ty CP kỹ thuật Hòa Phát.
- Thực nghiệm mô hình bằng Cisco Packet Tracer.

## Sản phẩm nộp

- Báo cáo thực nghiệm.
- Phiếu học tập cá nhân/nhóm.
- Phiếu phân công nhiệm vụ.
- File mô phỏng Cisco Packet Tracer.
- Hình ảnh minh họa mô hình mạng.
